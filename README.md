# 🐧 Jeu du Pingouin (Tux)

Un jeu Web interactif développé en JavaScript pur pour manipuler le DOM et gérer des événements en temps réel.

## 🎮 Fonctionnalités
- **Déplacement** : Contrôlez Tux avec les flèches du clavier.
- **Interaction** : Cliquez sur le cadre pour faire apparaître des "splats".
- **Score** : Système de score dynamique basé sur la précision des clics.



## 🛠️ Stack Technique
* **HTML5** : Structure sémantique.
* **CSS3** : Transitions fluides et mise en page.
* **JavaScript** : Logique de jeu, détection de collisions et gestion des événements (`keydown`, `mousedown`).
